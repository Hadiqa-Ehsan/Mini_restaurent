
# 🍽️ Mini Restaurant

A simple and interactive web-based restaurant menu and ordering system. Browse dishes, add items to your cart, and calculate your total bill.

## ✨ Features

- **Menu Display** – Showcases food items with prices and images
- **Categories** – Appetizers, Main Course, Desserts, and Beverages
- **Add to Cart** – Select multiple items with quantities
- **Cart Management** – Update quantities or remove items from cart
- **Bill Calculation** – Automatically calculates subtotal, tax, and total
- **Order Summary** – Review all selected items before placing order
- **Responsive Design** – Works on desktop, tablet, and mobile

## 🛠️ Technologies Used

- **HTML5** – Structure and layout
- **CSS3** – Styling, grid layout, and animations
- **JavaScript** – Cart logic, calculations, and dynamic updates

## 📁 Project Structure

```
Mini_restaurant/
├── index.html      # Restaurant menu and ordering interface
├── style.css       # Styling and responsive design
├── script.js       # Cart and billing logic
└── README.md       # Documentation
```

## 🔧 How to Run

1. Download or clone the project
2. Navigate to the project folder
3. Open `index.html` in your web browser

Or simply double-click the `index.html` file.
